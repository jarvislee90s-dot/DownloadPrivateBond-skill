# Ratingdog 下载细节

## 页面顺序

1. 打开 `https://www.ratingdog.cn/login`，使用手机号密码登录。
2. 进入 `https://www.ratingdog.cn/researchIssuer/yyRating`。
3. 在“代码、简称、发行人”输入公司名称并搜索。
4. 点击下方第一个匹配结果的发行人名称。
5. 在发行人详情页点击“相关公告”。
6. 对每只债券设置日期、私募筛选并下载附件。

## 公告筛选

- 从债券全称提取 `20xx` 年份。
- 日期范围设置为 `20xx-01-01` 至 `20xx-12-31`。
- 日期面板需要通过 Enter、Esc、blur 或页面点击收起后，再勾选“只看私募”。
- 搜索结果最多等待 30 秒，不使用固定长 sleep。

## 匹配和下载

- 标题匹配时先统一全角括号、半角括号、空格、引号。
- 只下载标题含“募集说明书”的附件。
- 不下载“发行公告”“发行结果公告”“法律意见书”“评级报告”等非目标文件。
- 点击下载后等待新 PDF 文件完成，确认没有对应 `.crdownload` 再继续。

## 常见失败

- WIND 返回 `0` 或债券全称没有年份：跳过并写入日志。
- “相关公告”刚出现但不可点击：循环等待可见且可点击，点击后再等待标题搜索框出现。
- 日期未生效：检查日期输入框值，确认先选日期再勾选私募。
- 没有年度私募匹配：撤销日期和私募，在全部公告中再查一次。

## 依赖处理

- 不在 skill 目录内安装 Python 依赖。
- skill 目录只放稳定脚本、固定模板、参考文档和示例配置。
- 运行环境需要已有 `openpyxl`、`selenium`、Chrome/ChromeDriver、WIND/Excel COM 环境。
- `win32com` 由本机 Excel/WIND 刷新步骤使用；缺失时不能执行真实 WIND 刷新。
- macOS 不支持完整 WIND 刷新流程；已有刷新后 Excel 时，可单独尝试 Ratingdog 下载脚本。
